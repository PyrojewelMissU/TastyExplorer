**遇到的问题**

TheMealDB API 单次搜索只返回约 60 个菜品
无法满足展示 200+ 菜品的需求


**解决方案**

① 使用 search.php?f={letter} 按首字母搜索
② 并行请求 20 个字母（a-z）
③ 使用 Promise.all() 等待所有请求完成
④ 用 Map 数据结构去重（以 idMeal 为 key）
⑤ 随机排序后取前 200 个菜品


**实现效果**

✓ 成功获取 150-250 个不重复菜品
✓ 请求耗时 6-8 秒
✓ 一次加载，永久缓存
✓ 再次访问瞬间加载

## 第 8 页：技术亮点 4

### 标题
技术亮点④ 跨组件状态同步

### 内容

**解决的问题**

详情页收藏后 → 首页和收藏页需要实时更新


**解决方案：storage 事件机制**

修改收藏时：
localStorage.setItem('favorites', JSON.stringify(favorites));
window.dispatchEvent(new Event('storage'));

监听收藏变化：
window.addEventListener('storage', loadFavorites);


**同步流程**

任意页面修改收藏
    ↓
更新 localStorage
    ↓
触发 storage 事件
    ↓
所有页面监听到事件
    ↓
自动更新显示状态


**实现效果**

✓ 收藏状态实时同步到所有页面
✓ 无需手动刷新
✓ 用户体验流畅自然

## 补充页（可选）：核心代码 1

### 标题
核心代码：批量导入

### 内容

```javascript
// 批量导入核心实现
const fetchFoodListFromAPI = async () => {
  const allMeals = new Map();

  // 选择 20 个首字母
  const letters = ['a', 'b', 'c', 'd', 'e', 'f', 'g', 'h',
                   'i', 'j', 'k', 'l', 'm', 'n', 'o', 'p',
                   'r', 's', 't', 'w'];

  // 并行请求所有首字母
  const requests = letters.map(letter =>
    axios.get(`${API_BASE_URL}/search.php?f=${letter}`)
  );

  // 等待所有请求完成
  const results = await Promise.all(requests);

  // Map 去重并合并
  results.forEach(meals => {
    meals.forEach(meal => {
      if (!allMeals.has(meal.idMeal)) {
        allMeals.set(meal.idMeal, meal);
      }
    });
  });

  // 随机排序，返回前 200 个
  const shuffled = Array.from(allMeals.values())
    .sort(() => Math.random() - 0.5);
  return shuffled.slice(0, 200);
};
```

```javascript
// 翻译系统核心实现
export const translateText = async (text) => {
  // 检查缓存
  const cacheKey = `en:zh:${text}`;
  if (translationCache.has(cacheKey)) {
    return translationCache.get(cacheKey);
  }

  try {
    // 尝试 MyMemory API
    let translated = await translateWithMyMemory(text);

    // 失败则尝试 LibreTranslate
    if (!translated) {
      translated = await translateWithLibre(text);
    }

    // 缓存结果
    if (translated) {
      translationCache.set(cacheKey, translated);
      return translated;
    }

    // 返回原文（优雅降级）
    return text;
  } catch (error) {
    console.error('翻译失败:', error);
    return text;
  }
};
```
